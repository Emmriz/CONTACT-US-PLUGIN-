Drag & drop contact form builder with multiple forms support and real-time preview.

== Description ==

Emmriz Contact Form is a powerful, user-friendly contact form plugin that allows you to create beautiful forms with a simple drag and drop interface.

= Features =

* **Drag & Drop Builder** - Easy visual form builder with real-time preview
* **Multiple Forms** - Create unlimited contact forms for different purposes
* **Real-time Preview** - See your form as you build it
* **Responsive Design** - Forms look great on all devices
* **Spam Protection** - Built-in honeypot protection
* **Email Notifications** - Get notified when forms are submitted
* **Submission Management** - View and manage all form submissions
* **Custom Fields** - Text, email, textarea, dropdown, and more
* **Shortcode Support** - Easy form placement with shortcodes
* **AJAX Submission** - Smooth form submission without page reload

= Usage =

1. Go to Contact Forms → Form Builder to create your first form
2. Drag fields from the left panel to the canvas
3. Configure field properties in the right panel
4. Use the shortcode `[emmriz_contact_form id="FORM_ID"]` to display your form

= Field Types =

* Text Input
* Email Input
* Textarea
* Dropdown Select
* Radio Buttons
* Checkboxes
* Number Input
* Submit Button

== Installation ==

1. Upload the `emmriz-contact-form` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Contact Forms → Form Builder to create your first form

== Frequently Asked Questions ==

= How do I add a form to my page? =

Use the shortcode `[emmriz_contact_form id="FORM_ID"]` where FORM_ID is the ID of your form.

= Where can I see form submissions? =

Go to Contact Forms → Submissions to view all form submissions.

= Can I create multiple forms? =

Yes! You can create unlimited forms for different purposes.

= Is the form builder mobile-friendly? =

Yes, the form builder works great on desktop, tablet, and mobile devices.
